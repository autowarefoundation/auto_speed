# auto_speed

## auto_speed_trainer.py

Main script for training auto_speed neural network

### Example usage

```bash
  python3 auto_speed_trainer.py --dataset /dataset_root_path
```

### Parameters:

*--dataset* : dataset directory path

*--input-width* : image input width

*--input-height* : image input height

*--batch-size* : batch size

*--local-rank* : local rank

*--version* : model version

*--runs_dir* : training runs directory

*--checkpoint_path* : pretrained checkpoint path

*--epochs* : number of training epoch
